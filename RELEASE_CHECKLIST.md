# Connexy アプリ リリース前チェックリスト

本番環境（プロダクション）へアプリを正式リリースする際、またはリリース後にユーザー数が増加したタイミングで確認・設定が必要な項目の備忘録です。

## 📍 Google Maps API (Places API) の利用上限解除
現在、開発中の意図しない課金を防ぐため、Google Cloudコンソール上で「1日あたりのリクエスト上限（Quotas）」を低めに設定しています。
アプリリリース後、案件投稿数が増えてきたタイミングで以下の対応を行ってください。

- **状態**: 現在、安全のために1日の利用上限（Requests per day）を制限中（例：300回/日など）
- **対応方法**: 
  1. [Google Cloud Console](https://console.cloud.google.com/) にアクセスする
  2. メニューから「APIとサービス」 > 「有効なAPIとサービス」 > 「Places API」を開く
  3. 「割り当て（Quotas）」タブから、「Requests per day」の上限値を引き上げる、または無制限に戻す
- **注意点**: 上限を開放すると、毎月の無料枠（$200）を超過した際に従量課金が発生するようになります。必要に応じて「予算アラート（Billing Alerts）」を設定しておくことを強く推奨します。
